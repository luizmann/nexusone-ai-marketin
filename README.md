## 🚀 The Complete AI-Powered Marketing Automation Platform

NexusOne AI is a comprehensive SaaS platform that empowers businesses to automate their entire marketing funnel using artificial intelligence. From product discovery to customer conversion, everything is automated and optimized for maximum ROI.

### 🌟 Key Features

- **🧠 AI Campaign Generator** - Create complete marketing campaigns in 60 seconds
- **🎥 AI Video Creator** - Generate professional videos with Luma AI, Runway, and more
- **📱 Magic Pages** - AI-powered landing page builder with drag & drop
- **🛒 Smart Dropshipping** - Automated product sourcing and fulfillment via CJ Dropshipping
- **💬 WhatsApp Automation** - AI chatbots for customer acquisition and support
- **📊 Facebook Ads Manager** - Automated ad creation and optimization
- **🎯 Lead Intelligence** - Smart CRM with predictive scoring
- **💰 Revenue Generator** - AI-powered income opportunity discovery

## 🏗️ Tech Stack

### Frontend
- **React 18** with TypeScript
- **Tailwind CSS** with custom design system
- **Shadcn/ui** components (40+ pre-built)
- **Framer Motion** for animations
- **Multi-language support** (EN, ES, PT, HE, AR)

### Backend
- **Supabase** (PostgreSQL + Edge Functions)
- **20+ Serverless Functions** for AI operations
- **Row Level Security** for data protection
- **Real-time subscriptions** for live updates

### AI Integrations
- **OpenAI GPT-4** for content generation
- **NexBrain Assistant** (Custom GPT Agent)
- **Luma AI** for video generation
- **ElevenLabs** for voice synthesis
- **Replicate** for image generation
- **Runway** for advanced video creation

### Marketing APIs
- **CJ Dropshipping** for product sourcing
- **Facebook Marketing API** for ad automation
- **Gupshup WhatsApp** for messaging
- **Unsplash** for stock imagery

## 🚀 Quick Start

### 1. Clone and Install
```bash
git clone https://github.com/your-username/nexusone-ai.git
cd nexusone-ai
npm install
```

### 2. Environment Setup
```bash
# Copy environment template
cp .env.example .env.local

# Add your API keys (see DEPLOYMENT.md for full list)
```

### 3. Development
```bash
# Start development server
npm run dev

# Open http://localhost:3000
```

### 4. Production Deployment
```bash
# Quick deploy to Vercel
npm run build
vercel --prod

# Deploy Supabase functions
supabase functions deploy
```
